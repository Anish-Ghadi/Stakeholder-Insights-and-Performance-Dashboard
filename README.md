# Stakeholder Insights and Performance Dashboard

## 📊 Project Overview
The **Stakeholder Insights and Performance Dashboard** is a data analytics project designed to analyze insurance business performance and stakeholder contributions.  

The dashboard provides insights into **premium collection, stakeholder value creation, profitability across cities, shareholding distribution, and asset management**.

This project helps organizations make **data-driven decisions** by identifying high-performing channels, stakeholder impact, and financial performance.

---

## 🚀 Dashboard Features

- Total Premium Analysis
- Stakeholder Age Analysis
- Equity Value Creation
- Value Created by Insurer Channels
- Premium Trends Over Time
- Profitability by City and Tenure
- Asset Under Management by Stakeholder
- Shareholding Distribution
- Cost Ratio vs Profitability

---

## 📈 Key Insights

### 1️⃣ Premium Performance
- Total Premium collected: **$245.91B**
- Indicates strong business growth and revenue generation.

### 2️⃣ Most Valuable Sales Channel
- **Bancassurance generates the highest value ($139.91B)**.

### 3️⃣ Stakeholder Demographics
- Average stakeholder age: **34.5 years**.

### 4️⃣ Shareholding Structure
- Foreign Institutional Investors (FII): **38.4%**
- Domestic Institutional Investors (DII): **30.2%**
- Promoters: **28.2%**
- Retail Investors: **3.3%**

### 5️⃣ Asset Under Management
- Individuals contribute the largest share in asset management.
